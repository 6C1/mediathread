Bookmarklet Import:

First install a browser "bookmarklet", and then use it to import items from supported collections and analyze them in MediaThread.

  2. Drag this link onto your browser bookmarks toolbar:
    Analyze w/Mediathread
    Google Chrome:

Bookmarklet Use:

Click the bookmarklet link when viewing an item in any supported collection to import the item to MediaThread. You can then analyze the item and/or save it to your "Items & Project" space.

What are Bookmarklets?

A bookmarklet is a tiny program (a JavaScript application) contained in a bookmark which can be saved and used the same way you use normal bookmarks. Bookmarklets run on the web page your browser is currently pointed at.

For more information see the Bookmarklet Wikipedia article
